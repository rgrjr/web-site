Home : Perl/Parrot : Advanced control structures : Conclusions

As we have seen:

  • Continuations are like an extended form of goto that can cross context boundaries.
  • We can use continuations to implement a wide variety of nonlocal control structures.
  • Such control structures extend the applicability of functional programming techniques.
